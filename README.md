# Dualword-wiki

Dualword-wiki - [Wikipedia](http://www.wikipedia.org) web browser (
[C++](http://www.isocpp.org), [Qt](http://www.qt.io), [WebKit](http://www.webkit.org) ).

Features:
 - access to Wikipedia website only
 - dual browser (dual language) tabs
